# Le Voyage de Crybaby

Une expérience immersive à travers les ères de Melanie Martinez.

## Comment lancer l'application localement

Si vous avez téléchargé ce projet depuis GitHub ou sous forme de fichier ZIP, vous ne pouvez pas simplement ouvrir `index.html` dans votre navigateur. Les applications modernes (Vite/React) nécessitent un serveur local pour fonctionner.

### Prérequis
- **Node.js** installé sur votre ordinateur (téléchargeable sur [nodejs.org](https://nodejs.org/)).

### Étapes à suivre :

1.  **Ouvrez un terminal** (ou une invite de commande) dans le dossier du projet.
2.  **Installez les dépendances** en tapant la commande suivante :
    ```bash
    npm install
    ```
3.  **Lancez le serveur de développement** :
    ```bash
    npm run dev
    ```
4.  **Ouvrez votre navigateur** à l'adresse indiquée dans le terminal (généralement `http://localhost:3000`).

## Technologies utilisées
- **React 19**
- **Vite**
- **Tailwind CSS**
- **Framer Motion** (pour les animations)
- **Three.js / React Three Fiber** (pour les environnements 3D)
- **Zustand** (pour la gestion de l'état)
